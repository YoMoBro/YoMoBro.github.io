Network troubleshooting — multi-VLAN repair

Diagnosed a broken multi-device network with mixed connectivity failures. Methodically tested pings between devices, audited running-configs, and identified three distinct issues: a misassigned VLAN, an incorrect subnet mask, and a missing static route. Restored full inter-device communication.

Configured RIP version 2 across multiple routers in a multi-network topology. Documented protocol behavior, subnet mask choices (/27, supporting 30 hosts per subnet), and verification steps. Compared dynamic vs static routing tradeoffs.

RIP v2 Dynamic routing Cisco IOS
